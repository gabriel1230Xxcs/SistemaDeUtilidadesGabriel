# Sistema de Utilidades

## Proyecto Final de Programación II

### Descripción

Sistema de utilidades desarrollado en C# Windows Forms que integra:

- Editor de texto.
- Paint con selección de colores.
- Calculadora funcional.

---

## Funcionalidades

### Editor de Texto
- Crear documentos nuevos.
- Escribir texto.
- Guardar archivos .txt en cualquier ubicación.

### Paint
- Seleccionar colores desde una paleta.
- Cambiar el color del panel en tiempo real.

### Calculadora
- Suma.
- Resta.
- Multiplicación.
- División.
- Validación para división entre cero.
